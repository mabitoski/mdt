Administration

Acces Microsoft

L'application accepte uniquement le SSO Microsoft. Un utilisateur doit appartenir a au moins un groupe MDT autorise pour ouvrir une session.

Microsoft only
Mode de connexion

Microsoft Entra ID uniquement. Les logins LDAP et locaux sont desactives.

Condition d'acces

Le compte doit etre present dans au moins un groupe MDT synchronise depuis Entra.

Reader

MMA-MDT-Readers : lecture seule sur le parc, les rapports et les PDF.

Operator

MMA-MDT-Operators : corrections atelier, commentaires et creation de report 0.

MMA-MDT-Logistics : lots, palettes et exports PDF groupes.

Platform admin

MMA-MDT-PlatformAdmins : acces a cette page et aux reglages sensibles.
